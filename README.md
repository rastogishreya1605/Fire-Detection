Real-Time Fire Detection System-

An AI-powered fire detection application that uses a computer vision-based approach to identify fire in real-time through a webcam. It features a professional full-screen interface and an automated audio alert system.

Key Features-

Real-Time Monitoring: High-speed frame processing for instant fire detection.
Intelligent Audio Alert: Plays a custom alarm.wav sound automatically when fire is detected.
Auto-Stop Mechanism: The alarm stops immediately as soon as the fire is removed from the camera's view.
Immersive UI: A clean, full-screen "Smart Mirror" style interface built with Streamlit, hiding all unnecessary headers and sidebars.
Visual Overlays: Large on-screen "FIRE DETECTED!" warning for immediate attention.
